# Mnemo

Self-hosted RAG over your Obsidian vault. Ask questions in natural language, get answers sourced from your own notes.

**Status:** 🚧 Phase 2 (RAG MVP) in progress.

## Architecture

Coming soon.

## Stack

- **Vector DB:** Qdrant
- **Embeddings:** Ollama + nomic-embed-text
- **Sync:** Self-hosted LiveSync (CouchDB) + livesync-bridge
- **API:** FastAPI
- **Language:** Python 3.11 + uv
